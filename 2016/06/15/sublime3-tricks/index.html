Sublime Text3 使用技巧

2016-06-15

常用快捷方式

开启和关闭

CTRL/⌘ + N 新建文件分页
CTRL/⌘ + O 快速打开当前资料架
CTRL/⌘ + W 关闭当前页
CTRL/⌘ + SHIFT + T 重新打开刚才关闭的一页

文档输入操作

CTRL/⌘ + SHIFT + V 根据缩进规则粘贴
CTRL/⌘ + / 快速注释/取消注释

CTRL/⌘ + D 选择当前光标范围内的字符串，连按可以依次连选之后的相同字符串

CTRL/⌘ + L 选择当前光标所在行，连按可以依次连选之后的行，同时按住SHIFT 键则在每行末出现光标

查找

CTRL/⌘ + F 在当前文档中查找

CTRL/⌘ + SHIFT + F 跨文件查找

CTRL/⌘ + R 文件爬虫，查找当前文档中的class类

CTRL/⌘ + SHIFT + P 快速更改设置，比如修改文档类型

其他窗口操作

CTRL/⌘ + SHIFT + 数字 分割成数字对应的窗口栏数

CTRL/⌘ + K + B (先按下K，再按下B) 关闭或打开侧边栏文件夹

⌘ + SHIFT + CTRL + F 当前页全屏

使用技巧

  • CSS 属性排序
    选中多行属性内容，按F5 ，就可以按字母顺序进行排列了

更多快捷键

Option + 鼠标滑拖 同时选中多行内容进行编辑操作
CTRL/⌘ + 鼠标点选 同时选中不连续的几行进行编辑操作

CTRL/⌘ + ←（ 或 →） 光标回到当前行最前或最后
SHIFT + ←（ 或 →） 连按，则逐个选择字符
CTRL/⌘ + SHIFT + ←（ 或 →） 以光标位置向前或向后选择当前行全部内容
