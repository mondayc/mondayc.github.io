我对BlockChain的理解

发布时间  2018-07-16        作者  Xiaoyu Ni        阅读         留言  0

好久没更新了，起初是觉得没啥好写的，时间长了也就搁置了。区块链作为一个新领域，自然需要去了解。计划看完五本书再来谈体会，可如果真等到那时候，黄花菜都凉了，更不会有动力了...遂决定每读一本书，就记录一次。待写完最后一篇，回顾心路历程，思路逐渐清晰，会真有收获吧。

在2017年之前，还未曾听说过区块链，却已见过比特币作为付款方式：其一是在微软中国官网；其二则是2016年夏天传播开的「永恒之蓝」Windows勒索病毒，（那时的一个比特币大概300美元，不到两年时间内，区块链概念热度就远高于同时期（2013社交APP，2014智能家居，2015可穿戴设备，2016虚拟现实，2017AI…）

什么是区块链

区块链是一个去中心化的分布式账本， 本质是一个P2P网络，由此决定了其不可被篡改的特性。然而去中心化很难被真正做到，用去中介化来表示或许更为合适。基础架构：数据层-网络层-共识层-激励层-合约层-应用层

中本聪

比特币的创始人，深藏功与名，一个精通密码学与经济学的技术黑客。

拜占庭将军问题

实用拜占庭容错机制及其改进算法为目前使用最授权多的联盟链共识算法。授权拜占庭容错机制最核心的一点，就是最大限度地确保系统的最终性，使区块链能够适用于真正的金融应用场景。

一些关键词

PoW：工作量证明。
不对称加密：公钥 私钥 （对称加密使用用户名密码）。
哈希值：对任何数据进行计算，都能得到一个固定长度的值。
公有链 私有链：公有链向所有人开放，私有链向特定的组织或个人开放。
智能合约：
信用共识：从基于道德 基于信仰 基于政府，到基于算法。

相关概念

  • 数字货币
    数字货币包括数字金币和密码货币，前者不能完全等同于虚拟世界中的虚拟货币，因经常被用于真实商品和服务交易，且不仅仅局限在网络游戏等虚拟空间中。

  • Currency Money Coin

我的理解

区块链的核心是分布式，而不是去中心化。而分布式的特点，也就意味着和物联网联系紧密。比特币解决的核心问题之一：时间排序，解决这个问题后，就能顺利的交换价值。想要靠炒币赚钱，不如直接开个交易所（皮一下很开心。

一些摘录

  1. 互联网交换的是信息，区块链交换的是价值。从信息互联网，到价值互联网。在前者，人们相互交换自己所产生的信息；区块链时代达成的机制：大家一起写一本书的N页，谁先把第N+1页的难题解决，谁就有权书写第N+1页。
  2. 区块链具有弱控制、分中心、自治机制、网络架构和耦合连接等等与工业社会完全不同的信息社会时代的新型社会结构、商业模式、人际关系。
  3. 区块链距离金融行业大面积应用还需数年时间。
  4. 「代码即法律」，只是乌托邦理想，智能合约既不是人工智能，也不是法律合同，只是一段帮助执行双方约定的计算机程序。
  5. 比数字货币范围更大、价值更大的是数字资产。

应用

书中举例了区块链在各个领域的应用场景，包括互联网安全管理及认证、能源领域等，其中能源方面包括电力、生态系统和智能调控，

  1. 去中介化自证清白的信任机制，而金融对信任的需求很高，因此在金融领域的应用水到渠成。OKLink搭建了基于区块链的全球汇款网络。
  2. 用区块链技术搭建医疗体系服务平台，以保证其安全性和可信度（Brontech）。
  3. 区块链让分布式的能源共享成为可能。区块链+物联网+大数据可以打造一个能源生态系统「乌托邦」。
